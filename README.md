# flask_bbs

Flask と SQLite を使ったシンプルな掲示板アプリです。  
投稿数が **1000件を超えると自動で全件削除** される仕組みになっています。

## Features
- Flask + SQLAlchemy を使用
- シンプルな掲示板形式
- 1000件を超えたら自動でリセット
- SQLite を利用した簡単なデータベース
